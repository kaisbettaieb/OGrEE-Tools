RackSearch(img: ndarray, customerAndSite: str, deviceType: str) → str

Reads a rack’s label on the input image and sends back all the data of the tenant,site,building,room of the rack and the rack itself as well as all the templates in use. If a step fails, catch the error and sends back its message

Parameters:

  • img (numpy.ndarray) – a numpy array containing an image in bytes

  • customerAndSite (str) – the name of the customer/tenant and the site where the rack is in the format {customer}.{site}

  • deviceType (str) – either “rack” or “mdi”

Returns:

a serialised dictionnary of serialised OgreeMessage’s instances if no Exception were raised, otherwise returns the exception’s message

Return type:

str

exception label_regular.Converter.IncorrectResponseError(url: str, endpoint: str, payload: dict[str, Any] | None = None, message: str = '')

Custom error class used when the API do not send back expected data

Parameters:

  • url (str) – the API’s url

  • endpoint (str) – the API’s endpoint

  • payload (dict[str,Any]|None) – the request’s payload. If None, the request will categorised as GET and as POST otherwise

  • message (str) – a custom error message. If precised, it will be printed instead of the default “GET/POST request to API {url} with endpoint {endpoint} returned incorrect data [payload :{payload}]”
